1. mvlrrl1.m implements the method proposed in this paper:
A cross-species bi-clustering approach to identifying conserved co-regulated genes
Jiangwen Sun, Zongliang Jiang, Xiuchun Tian and Jinbo Bi
Bioinformatics, 32 (12), i137-i146, 2016

2. sample_script_mvlrrl1.m is a sample script to run this method


3. nmi.m computes normalized mutual information (NMI) between two cluster solutions.

4. tbl.m calculates the distribution of values in the given discrete vector.
